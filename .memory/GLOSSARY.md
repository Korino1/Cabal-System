# Глоссарий
- Логическая схема: описание поведения агентов и фаз без кода.
- Автоцикл: повторяемая последовательность шагов выполнения протокола.
- Канон: источник истины с наивысшим приоритетом.
- Фаза: этап протокола с критериями входа/выхода.
- Снимок: фиксированное состояние (STATE.md) для восстановления.
