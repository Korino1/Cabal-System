# Zen4 (AMD) — Архитектурные заметки

Эта папка предназначена для `simd-specialist` и всех ролей, которые проектируют/реализуют микроядра, hot-loop, `asm`/intrinsics и регистровый план под Zen4.

## Что хранится в репозитории
- `zen4_registers.md` — быстрые заметки по регистрам/ABI/типовым ловушкам SIMD (это конспект, а не первоисточник).

## Что НЕ хранится в репозитории
Полные тексты vendor-документов (PDF/TXT-экстракты) здесь не коммитятся, если у них нет явного разрешения на распространение. Такие файлы должны скачиваться отдельно пользователем и храниться локально.

## Источники (скачать отдельно)
- AMD: *Software Optimization Guide for the AMD Zen4 Microarchitecture* (Doc ID 57647, Rev. 1.00, April 2023).
- AMD64 Architecture Programmer's Manual (системное программирование/ABI/инструкции).
- PPR/документация Family 19h (MSR/платформенные детали), если требуется.
