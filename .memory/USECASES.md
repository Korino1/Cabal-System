# Ключевые сценарии
1) Пользователь задаёт цель проекта → система задаёт порядок автоцикла → формируется последовательность фаз и обязанностей.
2) Происходит обрыв контекста → система восстанавливает состояние по STATE/GLOBAL_INDEX/PHASES.

## Acceptance Criteria
- После обрыва контекст восстанавливается без потери активной фазы и задач.
- Автоцикл не допускает перескока фаз и фиксирует шаги в WORKLOG.
