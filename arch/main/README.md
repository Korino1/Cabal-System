# arch/main (Agner Fog) — мануалы и инструменты для низкоуровневой оптимизации

Назначение: справочные материалы и инструменты для ролей, которые работают с `asm`/intrinsics, микроядрами и регистровым планом.

## Содержимое
- `calling_conventions.pdf` — соглашения о вызовах (calling conventions).
- `optimizing_assembly.pdf` — оптимизация ассемблерного кода.
- `microarchitecture.pdf` — обзор микроархитектур и факторов производительности.
- `asmlib.zip` — asm-библиотека Agner Fog.
- `objconv.zip` — objconv (конвертер/дизассемблер объектных файлов) + исходники.
